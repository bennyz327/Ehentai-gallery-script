# Ehentai-gallery-script

在 e-hentai 和 exhentai 畫廊頁面
直接管理收藏和選擇種子下載

Manage favorites and download torrents directly from the e-hentai and exhentai gallery pages

## 相比於原作者版本
- 種子區塊顯示更多資料提供使用者選擇依據
